Detailed Description

Klasa modelujaca g³owna czesc programu.

Klasa modeluje glowna czesc porgramu, ktorego zadaniem jest: -wczytanie danych -zmierzenie czasu dzialania algorytmu -sprawdzenie poprawnosci tego algorytmu, majac oczekiwany wynik
